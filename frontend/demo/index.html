🔌 API Endpoints

Base URL:

http://localhost:5000

Endpoints:

  • ✅ GET /health
  • 📊 POST /predict
  • 📍 POST /analyze-blind-spots
  • 🗺️ GET /heatmap
  • 🧪 GET /test

📝 Request Example

curl -X POST http://localhost:5000/predict \
  -H "Content-Type: application/json" \
  -d '{
    "latitude": 28.7041,
    "longitude": 77.1025,
    "crowdDensity": 50,
    "lighting": "moderate",
    "driverRating": 4.5,
    "historicalIncidents": 2
  }'
